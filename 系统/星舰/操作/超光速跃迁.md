# 超光速跃迁

## 介绍

超光速跃迁作为星际通航的核心技术，对于一艘星舰的航行而言至关重要。其允许你在星际之间快速穿梭，当然，其也需要相当长一部分时间来准备。

## 需求

执行超光速跃迁需要满足以下条件

- 用于充足的超光速燃料，并且舰体用于足够的能量储备供应超光速引擎。
- 必须拥有情况良好的超光速引擎，即未被损毁或处于故障状态。
- 超光速引擎准备期间引擎舱和驾驶舱必须有人操作。

## 执行

使用超光速引擎的流程

1. 驾驶台确认准备超光速引擎，选定目标信标。
2. 引擎舱校准超光速引擎。
3. 引擎舱报告完成准备操作。
4. 引擎舱与驾驶舱同时确认执行超光速跃迁。

此程序可能在不同的情况下有所区别，请以实际为准。

### 选择目标信标

受限于技术因素，超光速跃迁只能选择一个定位信标来进行，并且单次距离同时受到信标和飞船双重限制。

连续进行超光速跃迁会导致需要更多时间准备，因此远距离跃迁仍然很费时间。

### 校准超光速引擎

为了保证超光速的精确性，每次折跃前都需要乘员手动校准引擎。

校准引擎需要相当长时间，同时因为操作繁琐且需要反复等待引擎响应。

校准流程如下

1. 扔六次 d100 骰子，获得一个向量 α
2. 用当前飞船所处于信标的坐标 β 减去 α 得到偏移坐标 γ
3. 将目标信标坐标 σ 减去

## 效果

跃迁至目标信标。
