## 子能力

### 缴枪不杀

类型：主动能力

描述：只需要简单的宣传便可以挫败敌方抵抗到底的决心

限制条件

```
处于高度优势状态或更好
自身生命值不小于三分之二
```

激活效果

```
回合制状态时
	消耗 10ap
	消耗 20sp
	大幅提高敌方投降倾向
	大幅提高敌方逃跑倾向
```

### 战前动员

类型：主动能力

描述：一场好的动员可以提高作战效果

限制条件

```

```

```

		消耗 10ap
		己方全体获得额外 30 虚血
		己方在前三回合 敏捷 属性提高 10
```
